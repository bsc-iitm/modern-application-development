## Building locally

To work locally with this project, you'll have to follow the steps below:

1. Clone this project
1. Install everything in requirements.txt
1. Preview your project: `mkdocs serve`,
   your site can be accessed under `localhost:8000`
1. Add content inside docs folder
1. Commit and push
1. Once merged to master it will get auto deployed

Read more at [MkDocs](https://www.mkdocs.org/).
